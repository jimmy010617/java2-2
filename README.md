*기존 언어의 플랫폼 종속성
플랫폼 = 하드웨어 플랫폼 + 운영체제 플랫폼
프로그램의 플랫폼 호환성이 없는 이유
1. 기계어가 CPU마다 다름
2. 운영체제마다 API가 다름
3. 운영체제마다 실행파일 형식이 다름
(예시)
인텔 CPU를 가진 리눅스 환경에서 개발 -> C/C++ 응용 프로그램 개발 -> Intel CPU + 리눅스(실행)
                                     -> Apple 사의 MAC PC(실행되지 않음)
                                     -> Intel CPU + 윈도우 노트북(실행되지 않음)
